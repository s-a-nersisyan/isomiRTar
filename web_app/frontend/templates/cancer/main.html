TCGA-{{ cancer }} 5'-isomiR-gene interaction network

Nodes of the network are highly expressed 5'-isomiRs (included in the list of 5'-isomiRs contributing to 95% of total TCGA-{{ cancer }} miRNA sequencing reads) and their target genes. Edges represent regulatory interactions which are both sequence-based predicted (either by miRDB or TargetScan) and supported by a significant negative correlation of 5'-isomiR and gene expression levels (Spearman's correlation < -0.3, p < 0.05). Green nodes and edges stand for canonical miRNAs, red nodes and edges stand for non-canonical 5'-isomiRs and blue nodes stand for target genes.

TCGA-{{ cancer }} 5'-isomiRs overview

IsomiR target genes are predicted with the use of miRDB and TargetScan. Only miRDB predictions with Target Score ≥ 80 are counted. The number of TargetScan predictions is harmonized with miRDB by selecting the same amount of targets with the lowest cumulative weighted context++ scores. Predicted targets with non-detectable expression in TCGA-{{ cancer }} are not considered. We assume that 5'-isomiR downregulates target gene on mRNA level in TCGA-{{ cancer }} if their expression levels are significantly anti-correlated (Spearman's correlation < -0.3, p < 0.05).
